Живые worktree: !`git worktree list`

Ветки: !`git branch -vv`

Прочитай `.claude/forge.md` и командами оттуда собери: открытые задачи с метками,
задачи с метками `needs-decision` и `blocked`, открытые PR со статусом проверок.

Собери короткую сводку **для человека, который не читает код**:

1. **Требует твоего решения** — по каждой задаче с `needs-decision`: суть
   вопроса одним предложением, варианты, твоя рекомендация.
2. **В работе** — что делается сейчас; worktree без PR — признак упавшего агента.
3. **Ждёт мержа** — PR, готовые к `/land`.
4. **Бэклог** — сгруппируй по милстоунам-эпикам (`[M#-E#]`) и `area:*`,
   покажи 5 верхних задач, готовых к запуску.
5. **Аномалии** — задачи без меток, PR без задачи, ветки без PR, задачи старше
   недели без движения.

Без простыней. Списками, по делу.
